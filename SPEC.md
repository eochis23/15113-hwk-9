Your goal is to create a chess app using expo go.
There is a folder called skills that shows you how to do some things using this framework
The app is successful when the following game modes and features are implemented:
    - Game modes
        - Standard chess
        - Crazyhouse chess
        - Atomic chess
        - Duck chess
        - Checkers
    - Features
        - local storage of game history
        - cpu players of different difficulties for each game mode
        - Different timing modes for each game mode, including 1|1, 3|2, 10 minute, 15|10, and others
        - Player vs player for each game mode
        - multiple options for board and piece styles
        - move history for each game, updating live
        - game review where cpu moves are shown for each bad move the player makes to help them improve
